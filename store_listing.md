## Full Description
ReaderBookMark is a powerful bookmark manager and web reader built for manga fans, avid readers, and anyone who wants to organize their online reading lists.

**📚 Organize Your Library**
• Save links with categories: General, Manga, Book, Article, and Watchlist
• Track reading progress: Not Started, Reading, or Completed
• Auto-detect titles from URLs — manga titles are extracted automatically
• Display cover images for each bookmark
• Search and filter your library instantly

**🌐 Built-in Web Reader**
• Read manga, articles, and web pages directly in the app
• Built-in ad blocker filters common ad networks before they load
• Saves your scroll position automatically — continue where you left off
• Select any text for instant translation

**🔔 New Chapter Alerts**
• Background scanner checks for new manga chapters every 3 hours
• Supports popular Turkish manga sites out of the box
• Receive push notifications when new chapters are found
• In-app notification center with unread badge — never miss an update

**📤 Backup & Transfer**
• Export your entire library as JSON — share via file or clipboard
• Import from JSON backup to restore on any device
• Bulk add links by pasting multiple URLs at once

**🔄 Always Up to Date**
• Check for app updates directly from the app
• Download and install new versions with one tap
• Secure updates with SHA-256 verification

**🎨 Modern Experience**
• Material 3 design with smooth animations
• Dark mode support
• Available in Turkish and English
• "What's New" screen after each update highlights new features

**🎬 Watchlist**
• Track movies, series, and videos alongside your reading library
• Dedicated category filter to quickly find your watchlist items

---

Whether you're following the latest manga chapters, building a reading list, or organizing video watchlists — ReaderBookMark keeps everything in one place.

## Privacy Policy
This application stores all user data locally on the device. No personal data is collected or transmitted to external servers. The only network requests made are:
- Fetching web pages the user explicitly opens
- Checking for new manga chapters from user-configured sites
- Checking for app updates from GitHub Releases
